# Machine-Learning-from-Scratch
A multimodal traditional process machine learning project with Snowflake that builds a streamlit UI.

To build the demo project first access the [Quickstart Guide](https://www.snowflake.com/en/developers/guides/first-machine-learning-project/) and download the notebooks [HERE](https://github.com/Snowflake-Labs/snowflake-demo-notebooks/tree/main/First_Machine_Learning_Project) and fork the repository that has the bear dataset [HERE](https://github.com/dataprofessor/bear-dataset/).
